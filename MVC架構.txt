MVC 架構

View (JSP 視圖層)
register.jsp <- 會員註冊頁面
login.jsp <- 會員登入頁面
profile.jsp <- 會員資料頁面
member-list.jsp <- 會員全部資料頁面(管理者才能閱讀)
error.jsp <- 錯誤處理頁面

View 職責:
顯示畫面
接收使用者使用資料
顯示 Controller 傳來的資料

-----------------------------------------------------
Controller (Servlet 控制層)
RegisterServlet <- 會員註冊處理
LoginServlet <- 登入驗證 + Session 建立
ProfileServlet <- 個人資料獲取(需登入)
UpdateServlet <- 修改個人資料(需登入)
MemberListServlet <- 管理者查看全部會員資料(需登入 + 管理員角色)
LogoutServlet <- 登出

Controller 職責:
接收 request
處理流程控制 (forward / redirect)
呼叫 Dao
判斷登入 / 權限


-----------------------------------------------------
Model(資料模型層)
entity.Member <- 會員資料 Entity
dao.MemberDao <- 資料操作(CRUD: Create/Read/Udate/Delete)
util.DBUtil <- 資料連線工具程式
MySQL <- 資料儲存層(MySQL資料庫)

Model 職責:
管理資料結構
管理資料庫存取

-----------------------------------------------------
MySQL 資料庫建置

create database educenter character set utf8mb4

-----------------------------------------------------
MySQL 資料表建置

use educenter;

create table if not exists member (
	id int primary key auto_increment,
    username varchar(50) unique not null,
    password varchar(500) not null,
    fullname varchar(50) not null,
    email varchar(100) not null,
    role varchar(50) not null,
    create_time datetime default current_timestamp
);
